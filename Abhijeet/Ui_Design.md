
# JusticeConnect Legal Aid Portal & Lawyer Case Management Dashboard

## Project Overview

This project includes UI designs for a **Legal Aid Portal** and a **Lawyer Case Management Dashboard**.
The system is designed to help users request legal assistance online and allow lawyers to manage cases efficiently.

## Features

### User Side

* Simple and user-friendly interface
* User registration and secure login
* Submit legal cases online
* Track case status

### Lawyer Dashboard

* View and manage legal cases
* Track case progress
* Manage client information
* Organize legal documents

---

## Purpose

The goal of this project is to **digitize legal assistance services** and provide an easy platform where users can request help and lawyers can manage cases in a centralized system.
