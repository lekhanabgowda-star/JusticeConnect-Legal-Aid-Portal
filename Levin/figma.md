## Features
- Modern, minimal, and responsive design supporting both light and dark modes across desktop, tablet, and mobile devices.
- Consistent icon-based navigation with a clear visual hierarchy using cards, tabs, and color-coded status indicators.
- Notification and feedback messages to keep users informed about actions and system updates.
- Key pages include **Dashboard, Case Submission, Lawyer Matching, Case Tracker, Appointments, and Documents**.
